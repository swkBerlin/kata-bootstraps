# Java Truth

A minimal setup with Java, [JUnit5](https://junit.org/junit5/) and [Truth](http://google.github.io/truth/) to get you started.

You can use [gradle](https://gradle.org/) or [maven](https://maven.apache.org/)

## Setup

    git clone https://github.com/swkBerlin/kata-bootstraps
    cd java/truth

Open as preexisting project in your favorite IDE and choose between gradle or maven nature

## (optional) IDE Setup

Run `./gradlew idea` to build [idea](https://www.jetbrains.com/idea) project.

## Running Tests

To execute the tests either run `./gradlew test`, `mvn test` or run the tests from the IDE you are using

## Test Libraries Available from the Get-Go
- [JUnit 5.7.2](https://junit.org/junit5/docs/snapshot/release-notes/#release-notes-5.7.2)
