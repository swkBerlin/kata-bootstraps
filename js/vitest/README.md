# Vitest unit tests

This folder contains a unit test template for [Vitest](https://vitest.dev).

Supports ES6 syntax out of the box.

## Installation

```bash
npm install
```

## Running the tests

```bash
npm test
```

Or you can use built-in watch mode:

```bash
npm run test:watch
```

## Show coverage

```bash
npm run test:cover
```

Note: coverage is also included in watch mode.
